
The folder `webgraph-openapi` contains the code that implements a web interface on the `WebGraph` framework to allow the user to explore big graphs dynamically.

It is the snapshot of [an upstream repo](https://github.com/massimo-nocentini/webgraph-openapi) at commit [0a75320](https://github.com/massimo-nocentini/webgraph-openapi/commit/0a75320d416015e68dfb269d31d552bd0d022d05).
