# Bitcoin Graph Models

This repository contains code for the paper *"Systematizing Graph Models for Bitcoin and Designing Algorithms for Large-Distance
Analysis"* by Anna Bernasconi, Damiano Di Francesco Maesa, Matteo Loporchio, Andrea Marino, Massimo Nocentini, and Laura Ricci.

The paper is currently under review and the code is provided here for reproducibility purposes. 

## Data availability

For space reasons, the data used for the experiments is not included in this repository. However, such data can be obtained from the following Zenodo repository: 

https://doi.org/10.5281/zenodo.18938800 

Please note that all downloaded data should be placed in the `data` folder of the repository and decompressed.

## Technologies used

For the data analysis process, we used the following languages and libraries:

- Bash
- C++
    - igraph (https://igraph.org/) 
- Java
    - WebGraph (https://webgraph.di.unimi.it/)
- Python 
    - Matplotlib (https://matplotlib.org/)
    - Numpy (https://numpy.org/)
    - Polars (https://pola.rs/)
    - Scipy (https://scipy.org/)
